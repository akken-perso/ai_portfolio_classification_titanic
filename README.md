# ai_portfolio_classification_titanic
Ce projet a pour but de prédire la survie d'un passager. Il est fondamental car il vous force à gérer des données catégorielles et des valeurs manquantes complexes, et à travailler avec des métriques de classification (plus complexes que la régression).

Objectif : Prédire si un passager a survécu (Survived: 0 ou 1).

Jeu de Données : Kaggle Titanic (fichiers train.csv et test.csv).
